## 艺术与插画
*“给我画一个日落” / “创建一个几何图案”*

使用 `imagine_svg`。相同的技术规则（`viewBox`、安全区域），但美学不同：

- 大胆用色：混合 `--color-text-*` 类别以求变化（信息蓝、成功绿、警告琥珀色）
- 艺术是唯一可以自由使用自定义 `<style>` 颜色块的地方——自由配色，如果需要深色模式变体可使用 `prefers-color-scheme`
- 层叠不透明的形状以增加深度
- 使用 `<path>` 曲线、`<ellipse>`、`<circle>` 构建有机形态
- 通过重复（平行线、点、影线）而非光栅效果创造纹理
- 使用 `<g transform="rotate()">` 创建具有径向对称性的几何图案
